## Deployment
This website is deployed on Vercel and can be easily deployed by:
1. Connecting your GitHub repository to Vercel
2. Or using the Vercel CLI with `vercel` command

## Local Development
Simply open `index.html` in a browser or serve with any local server:
```bash
npx serve
# or
python -m http.server 8000
```
